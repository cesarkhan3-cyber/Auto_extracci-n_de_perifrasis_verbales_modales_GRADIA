# Auto_extracci-n_de_perifrasis_verbales_modales_GRADIA
Este repositorio ofrece un script de python que ayuda a llevar a cabo una extracción automática de 6 perífrasis verbales modales ("deber + INF", "deber de + INF", "tener de + INF", "tener que + INF", "haber que + INF" y "haber de + INF") desde archivos de textos planos (.txt). Se adjunta también resultados de un análisis colostruccional.


pip install -r requirements.txt


python -m spacy download es_core_news_md
